Packed with features that delight users

Rows & Columns Interactions

The Data Grid provides a rich inline editing experience so your users can easily update information in the grid. Make the grid editable with a property configuration, add one of the provided cell editors or easily create your own. Full row editing is also available. Need to change the order of the displayed records or columns in the data grid? Easy! Your users can Drag & Drop columns to position rows and columns as required.

Business Intelligence & Data Exploration

Being able to quickly analyze data helps your users make better business decisions. Our React grid makes it easy to summarize, analyze, and explore data in just a few clicks. Sort and Filter the data. Use the grouping/aggregation and pivoting functionality to group by specific columns. Apply various aggregate functions for row and column totals. Select the data in the grid and create charts to visualize and present.

Rich UI controls

Users expect a modern, accessible and familiar GUI. We provide you with a grid that's easy to style with CSS, or use pre-built styling themes. Accessible and configurable to work with the same interactions as a desktop spreadsheet. Customizable columns and context menus make it easy to filter and copy/paste records, resize and pin columns. An intuitive GUI that works the way your users expect and easy to customize for your application's specific needs.

Hierarchical Views & Nested Grids

Business data has relationships between entities. Our React data grid implements a tree view specifically designed to display parent / child relationships between entities. We also support nested grids to render child data. Expand and collapse groups in one click. And when you need to perform some analysis, the data aggregation and filtering functionality is available for users even in the hierarchical views.
